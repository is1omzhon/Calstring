# Calstring - String asosida hisoblash dasturi

## 📌 Loyiha haqida
Bu dastur foydalanuvchi kiritgan matnli ifodani (masalan "1+2-3+4") hisoblaydi.
Dastur faqat **string indekslari** va **while loop** yordamida ishlaydi.

## 🚀 Qanday ishlaydi?
Foydalanuvchi misol: `1+2-3+4` kiritadi
Dastur:
1. Birinchi sonni oladi (1)
2. Amalni va keyingi sonni oladi (+, 2) → 1+2=3
3. Amalni va keyingi sonni oladi (-, 3) → 3-3=0
4. Amalni va keyingi sonni oladi (+, 4) → 0+4=4
5. Natijani chiqaradi: 4

## 📝 Qo'llab-quvvatlanadigan amallar
- `+` Qo'shish
- `-` Ayirish
- `*` Ko'paytirish
- `/` Bo'lish
- `%` Qoldiq
